Пример запуска:
```shell
python ./main.py

Если не указаны действия вернет краткую сводку о файле:
python ./csv.py ./filename.csv

Список действий:
    -i, --info - показать краткую сводку о файле
    -rd, --remove_duplicates - удалет строку при дублировании в указанной колонке
        Если номер колонки не указан берется первая колонка. Номер колонки для обработки 
        указывается в ключе -cn --column_number, если не указано нумерация колонок 
        начинается с 0.
        Пример:
            python ./csv.py ./tests/test_files/test3.csv -rd    
```
